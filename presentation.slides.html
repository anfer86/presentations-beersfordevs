O Que São Modelos?¶

Um Exemplo

Uma empresa que vende cervejas possui milhares rótulos. Obter opiniões confiáveis de clientes é custoso. A empresa fez um investimento para obter 300 avaliações de clientes sobre diferentes cervejas. No sistema de avaliação de produtos os clientes avaliam cada cerveja com as opções: Compraria ou Nao Compararia.

Problema

A empresa gostaria de saber se dadas as características de uma cerveja, é possível estimar se seria comprada pelos clientes ou não.

Coleta de Dados

Das cervejas avaliadas pelos clientes a empresa coletou as características: Alcool (teor alcoólico) e IBUs (unidades de amargor).

Em Aprendizado Supervisionado, os modelos são funções que tentar predizer um objetivo.

Aprendizado supervisionado é tipo de técnica de Machine Learning usada quando temos instâncias rotuladas do que queremos prever. Por exemplo, a empresa tem 300 instâncias de cervejas avaliadas pelos clientes.

Modelagem

Considere $x_1 = \text{Alcool}$ e $x_2 = \text{IBUs}$. Vamos ajustar um modelo tem o seguinte formato:

$$ f(x_1, x_2) = w_1 * x_1 + w_2 * x_2 + b$$

O modelo vai assumir que:

  • Os valores Negativos se referem a Compraria
  • Os valores Positivos se referem a Não compraria

O valor de $w_1$ é o peso do Alcool na escolha de compra.

O valor de $w_2$ é o peso de IBUs na escolha de compra.

Procurando uma Boa Solução

In [84]:
display(ui, out)

Um método chamado de Regressão Logística consegue encontrar os melhores valores para essa função:

  • $w_1 = 0.53$
  • $w_2 = 0.25$
  • $b = - 6.70$

Assim, o modelo para predizer se os clientes vão comprar uma cerveja ou não é dado por:

$$ f(\text{Alcool}, \text{IBUs}) = 0.53 * \text{Alcool} + 0.25 * \text{IBUs} - 6.70 $$
  • Valores Negativos se referem a Compraria
  • Valores Positivos se referem a Não compraria

Texto, Imagens, Dados Temporais, Trajetórias¶

Estes tipos de dados estão super na moda atualmente. Vejam como as coisas se complicam:

- Texto: e se cada cliente que avaliou uma cerveja ao fez um comentário próprio de até 400 caracteres sobre a mesma? e se fez uma postagem nas redes sociais a respeito?¶

- Imagens: e se tivessemos accesso ao rosto do cliente quando ele experimentou a cerveja? ou uma foto na postagem da rede social? qual sentimento poderia ser extraído?¶

- Dados Temporais: e se soubéssemos o histórico (marca, tipo, ...) de consumo de cervejas desses clientes?¶

  • Trajetórias: e se soubéssemos os locais e horários em que o cliente bebe cerveja? o que poderíamos oferecer? será que eles bebem após trabalhar ou estudar, quando viajam para outra cidade?
